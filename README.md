# 🍹 LDR Smoothie Cashier System

Sistem Informasi Kasir Digital yang dirancang khusus untuk manajemen penjualan, pencatatan transaksi, dan pengelolaan stok inventaris pada bisnis **LDR Smoothies & Juice**. Sistem ini dibangun dengan arsitektur terpisah antara komponen *Frontend* dan *Backend* untuk memastikan performa yang optimal dan kemudahan pengembangan.

---

## 🚀 Fitur Utama
* **Manajemen Transaksi Kasir:** Pencatatan pesanan produk dengan pilihan ukuran porsi yang akurat (Ukuran K untuk Kecil, B untuk Besar pada varian Jus, dan L untuk porsi Large pada Smoothies).
* **Manajemen Stok & Inventaris:** Pemantauan sisa bahan baku dan stok buah secara real-time untuk mencegah kekurangan persediaan.
* **Dashboard Admin & Laporan Penjualan:** Visualisasi data transaksi harian, mingguan, dan bulanan untuk membantu analisis bisnis.
